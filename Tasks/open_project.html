Opening an Existing Project

To open a project in DITAxPresso, follow the steps below.
  1. Click Open Project in the workspace.
    Open Project dialog box is displayed.
  2. In the dialog box:
    1. Click Choose and browse for DITAxPresso project with .dxp extension.
    2. Select the required file with .dxp extension and click Open.
    3. Click Open Project.
The project files are listed under the Project Manager on the left-hand pane of the window.
