Style Frontpage

This task explains how to format the front page for a customized output.
  1. Open a DITAxPresso project.
  2. Click Styler > PDF Styler > Front Page.
  3. If you want to add logo to the front page, follow the steps below.
    1. Click LOGO.
    2. Click Browse. Navigate to the logo.
    3. Select logo and then click Open.
    4. Select alignment.
    5. Select the size.
  4. Click TITLE. Choose suitable options for the fields you want to customize.
  5. Click SUBTITLE. Choose suitable options for the fields you want to customize.
  6. Click Save.
