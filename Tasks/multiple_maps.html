Selecting from Multiple Maps

Multiple DITA maps can be present in a folder.

To select one DITA map from multiple DITA maps in the project:

  1. Open a project. Opening an Existing Project
    Note: In the workspace, a message will appear Multiple DITA maps detected in the project directory. Select the required DITA map.
  2. Click the drop-down list arrow following the message .
  3. From the list box, select the required DITA map.
The DITA map structure is displayed under the DITA Map Viewer on the left-hand pane of the window.
