Customize XSLT for PDF

The PDF Styler allows to customize the output based on the user requirements. The user has the privileges to modify or design the document based on the branding strategy.

Below are the various properties to customize the PDF output:

  • Style Frontpage
  • Style Page Header and Footer
  • Style TOC
  • Style Tables
  • Style Lists
  • Style Commons
