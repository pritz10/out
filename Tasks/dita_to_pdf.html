DITA Map to PDF

This task explains how to generate PDF files using DITA Map files.
  1. Open the required DITA map in the DITAxPresso project.
  2. From the left-hand pane of the window, click Publisher > PDF.
  3. In the workspace, Selecting Plugin.
    Note: To style the PDF before publishing, Customize XSLT for PDF.
  4. Click Transform.
In the project folder, PDF file is generated in Output > pdf.
